# FolderZipper
Recursively compress files and folders

```
Usage:
        -s, --sourceFolder... Source Folder

        -d, --destinationFolder... Destination Folder

        -t, --textFile[optional]... Create a textfile with all the files in a folder.

        -u, --summaryTextFile[optional]... Create a summary textfile with all the files.

        -p, --showPogress[optional]... Show progress in %.

        -o, --overwrite[optional]... Overwrite destination files

        -c, --compression[optional]... CompressionLevel (Level0;None;BestSpeed;Level1;Level2;Level3;Level4;Level5;Default;Level6;Level7;Level8;BestCompression;Level9)
```
